🏏 IPL SQL Analysis – Royal Challengers Bangalore (RCB)

📌 Project Overview

This project analyzes the performance of Royal Challengers Bangalore (RCB) in the Indian Premier League (IPL) using SQL. The analysis answers various business and sports-related questions by querying an IPL relational database.

The project demonstrates SQL skills including joins, aggregations, filtering, grouping, subqueries, and data analysis techniques.

---

 🎯 Objectives

- Analyze RCB's overall performance
- Calculate total runs scored
- Identify top run scorers
- Evaluate match outcomes
- Analyze venue-wise performance
- Generate insights using SQL queries
